FAQ do Atendente PNCP

Alteração de UASG?

A alteração de UASG é realizada pelo sistema SGA. Ao receber esse tipo de chamado, direcione-o para a estruturante correta.

Erro ao enviar uma contratação por inexigibilidade para o PNCP?

Geralmente, esse erro ocorre no sistema Compras.gov.br. Verifique no chamado se o usuário adicionou alguma evidência do portal que está utilizando. Se não, peça essa informação. Se o sistema for o Compras.gov.br, direcione o chamado para a estruturante correta.

Informações erradas no sistema PNCP?

Saiba que a responsabilidade da manutenção e divulgação dos dados é do sistema que o usuário está utilizando. Verifique qual é o sistema do usuário. Se for o Compras.gov.br, direcione para a estruturante correta. Se for um sistema privado, vá até a página "script", onde você encontrará a mensagem para essa situação.

Erro - compra não foi publicada no PNCP

É necessário verificar qual é o sistema que o usuário utiliza para divulgar as informações, pois o sistema é responsável pela manutenção dos dados divulgados no PNCP. Se a publicação não está disponível no PNCP, verifique o sistema. Caso seja o Compras.gov.br, direcione o chamado para a estruturante correta. Se não, solicite que o usuário entre em contato com os desenvolvedores do sistema dele. Se for um desenvolvedor que está entrando em contato, vá até o Guia de Atendimento e solicite as informações técnicas.
